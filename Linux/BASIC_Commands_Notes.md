List of learned commands:

clear- clear the terminal window

reset- reset terminal settings

whatis- one-line help for commands

Key --help - Brief help on commands

man, info- detailed help on commands

apropos- search by reference

history- output command history


pwd- (print working directory) print the path to the current folder

cd- (change directory) change folder

ls- display the contents of the folder

ll- a nickname forls -la


mkdir- (make directory) create a new folder

tree- display the file structure as a tree

touch- create file

echo- display text on the screen (or send to a file)

cat- (concatenate) output of "glued" contents of files


cp- (copy) copy a file or folder

mv- (move) move/rename a file or folder

rm- (remove) to delete a file or folder


more- page-by-page viewing of file contents

less- page-by-page viewing of file contents

head- view the first lines of the file

tail- view the last lines of the file

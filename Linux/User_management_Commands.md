**List of learned commands:

whoami- display username (login)

id- display UID, GID, login, list of additional user groups

groups- display a list of user groups

su- (substitute user) - change user

sudo- (super user do) execute a command as a superuser



useradd- create user (main command)

adduser- create user (perl script)

passwd- change the user's password



usermod- change user

usermod -L- block user

usermod -U- unblock user



visudo- edit sudoers file (access rights)

userdel- delete user



groupadd- create a group

addgroup- create a group

groupdel- delete group**


List of learned commands:

top –o PID- run the top utility sorted by the PID field

top –u salesguy- run the top utility with filtering of processes by the user salesguy



sudo apt-get install htop- install the htop utility

ps- display information about the current process

ps 1 - display information about process number 1

ps aux- display a snapshot of the state of all processes in a table

ps afx- display a snapshot of the state of all processes in the form of a tree

pidof - find process number by its name

kill - complete the process

systemctl - utility for managing services

systemctl status ssh- display information about the state of the SSH service

systemctl stop ssh- stop SSH service

systemctl start ssh- start the SSH service

systemctl is-active ssh- check: is the SSH service running
